# Dynamic Alignment Observatory

Mapping diplomatic and trade alignment across the international system. A collaboration between Shahryar Minhas (Michigan State University) and the Center for Strategic and International Studies (CSIS).

**Dashboard**: https://csisalignment-nsbqvwiqgdsfirzruqgks4.streamlit.app/

## What Is in the Repository

All R scripts, Python dashboard code, output CSVs, static figures, the memo, and the dashboard data files are tracked in git.

## What Is Excluded (gitignored)

The following are excluded for size reasons. They are regenerated by running the R pipeline.

| Path | Contents | How to regenerate |
|------|----------|-------------------|
| `data/raw/` | Downloaded source data (Voeten UNGA scores, ~21 MB) | Run `R/02_ingest/01_un_voting.R` |
| `data/processed/*.rda` | Cleaned dyadic data (UNGA, alliances, trade) | Run `R/02_ingest/` scripts |
| `data/networks/*.rda` | 4D arrays for modeling (193 x 193 x 3 x 60) | Run `R/02_ingest/09_build_networks.R` |
| `results/models/*.rda` | Fitted LAME and DBN model objects | Run `R/03_models/` scripts |
| `*dyad_year_unga_agreement.csv` | Raw UNGA agreement scores (101 MB) | Run `R/05_analysis/01_descriptive.R` |

## Data Sources

| Source | Coverage | Used for |
|--------|----------|----------|
| Voeten UNGA voting (Harvard Dataverse) | 193 countries, 1965-2024 | Diplomatic alignment |
| ATOP alliance data (from Carbon project) | 128 countries, 1965-2018 (carried forward to 2024) | Network array input |
| IMF Direction of Trade Statistics (from Plutonium project) | 140 countries, 1990-2020 | Trade alignment |
| World Bank WITS API | 186 countries, 2021-2023 | Extends trade to 2023 |

## Running the Pipeline

Requires R 4.3+ with packages: `tidyverse`, `countrycode`, `netify`, `lame`, `dbn`.

```bash
# Full pipeline (takes ~3 hours for the 193-country LAME model)
cd csis
Rscript R/01_registry/01_build_actor_registry.R
Rscript R/02_ingest/01_un_voting.R
Rscript R/02_ingest/02_alliances.R
Rscript R/02_ingest/03_trade.R
Rscript R/02_ingest/03b_trade_wits.R
Rscript R/02_ingest/09_build_networks.R
Rscript R/03_models/02_lame_benchmarks.R
Rscript R/04_fusion/03_extract_full_scores.R
```

## Running the Dashboard

```bash
cd csis/app
pip install -r requirements.txt
streamlit run app.py
```

Or with Docker:

```bash
cd csis/app
docker build -t csis-alignment .
docker run -p 8501:8501 csis-alignment
```
